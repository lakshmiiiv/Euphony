# E U P H O N Y 🎧
**B.Tech IT - 3rd Year Final Project**

### 🚀 Key Features
* **ML Mood Engine:** Uses Euclidean Distance for real-time playlist generation.
* **Euphony Blues:** A gamified loyalty system with 5x reward multipliers.
* **Find My Squad:** Secure student social networking with institutional verification.
* **Quick Attendee:** Indexed NoSQL lookup for 1-click ticket booking.

### 🛠️ Tech Stack
* **Frontend:** Flutter (Dart)
* **Backend:** Firebase Realtime Database
* **API:** iTunes Search API
